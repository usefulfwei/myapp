## myapp

安装mongodb、node（6.2.0以上）

- 安装
    - git clone git@github.com:u14e/myapp.git
    - cd myapp/
    - npm install

- 开发模式
    - 运行mongod
    - 终端1：npm start
    - 终端2：npm run dev
